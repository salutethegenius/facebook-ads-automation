Welcome to Facebook Ads Automation

This tool helps you automate the creation of Facebook ad campaigns, including targeting and ad generation.

What You Can Do:
  • Create a campaign with a specified budget
  • Set up an ad set with targeting parameters
  • Create an ad creative with text
  • Generate an ad

Note: All components are created in PAUSED state to prevent accidental spending.

Getting Started

To use this tool, you need to authenticate with the Facebook Ads API. Please provide your credentials below:

You can get this from Facebook for Developers.

Prerequisites

Before you begin, make sure you have:

  • A Facebook Developer Account
  • An app registered at Facebook for Developers
  • Access token with ads_management permissions
  • Ad Account ID
  • Facebook Page ID (for the ad creative)
